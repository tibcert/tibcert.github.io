Services

These are some of the services which will be provided to a member organization. The services are outlined below:

Digital Security Policy Development & Implementation

Guidance of secure tools & infrastructure deployment

Digital Security Incident Response

Training on Digital Security

Organizational Security Audit

Response Working Group

The Response working group would consist of the following members:

Tibet Action Digital Security Team

Tibet Action Digital Security Ambassadors

A contact person from each member organization

Tibetans working in IT

Supporting organizations - Ex. Citizen Lab

Any new member who is not part of an organization can be introduced by another member
